# PascalABC-WPF-Xaml-Designer
PascalABC.net 개발툴을 활용한 pascal wpf xaml Designer

[Ver 1.2.4]

* 빌드 및 오류 관련 모듈 수정

* 빌드 하면 콘솔창에 '[0][1,6] MainWindow.pas: Unit name must match the file name' 메세지가 표시되는데 빌드는 되고
  
실행하면 빌드할때처럼 같은 콘솔창 메세지 표시되고 MainWindow.exe가 실행되기는 하는데 MainWindow.xaml 의 디자인과는 다르게 윈도우 창 1개만 나타남.
