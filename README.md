# PascalABC-WPF-Xaml-Designer
PascalABC.net 개발툴을 활용한 pascal wpf xaml Designer

[Ver 2.0.0]

* VS 호환 XAML 지원

 x:Class, x:Name, 이벤트 속성 100% 호환
 
 mc:Ignorable, d:DesignHeight/Width 지원


* 프로젝트 템플릿

 WPF 애플리케이션 (Window 루트)
 
 WPF 사용자 정의 컨트롤 라이브러리 (UserControl 루트)


* 자동 코드 생성

 InitializeComponent() 자동 생성
 
 FindName() 컨트롤 바인딩
 
 이벤트 핸들러 시그니처 자동 생성

* 주의사항

 MainWindow.exe 의 정상적인 작동을 위해 같은 폴더에 MainWindow.xaml 파일이 존해해야함.
 
 그외 소소한(?) 버그가 있을수도 있음.
