# PascalABC-WPF-Xaml-Designer
PascalABC.net 개발툴을 활용한 pascal wpf xaml Designer

[Ver 1.2.0]

* 빌드 기능을 추가하면서 UI 많이 변경됨
  
* 기존 폼디자이너 아래에 xaml 에디터 있는 구조에서 폼디자이너탭/xaml 에디터탭/pascal 소스코드 탭 으로 변경 

* 실제 빌드는 "C:\Program Files (x86)\PascalABC.NET\pabcnetc.exe" 여기서 빌드함.

* 아직은 빌드기능이 제대로 작동안됨에 주의.

* LoadXaml 문장 수정

* 오류목록 리스트 추가 했는데 cmd pipeline 과의 연결에 문제가 있는지 작동이 안됨.
