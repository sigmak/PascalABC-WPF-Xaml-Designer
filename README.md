# PascalABC-WPF-Xaml-Designer
PascalABC.net 개발툴을 활용한 pascal wpf xaml Designer

[Ver 1.2.5]

* WPF 코드비하인드 템플릿 생성

* 코드 탭에서 사용하는 클래스명 접두사를 가져오는 헬퍼
* 코드 파일에 이벤트 핸들러 추가 모듈 수정
* 빌드시 pabcnetc.exe 는 콘솔 핸들을 직접 사용하므로 stdout 리디렉션 불가.

     bat 파일을 통해 UseShellExecute=true 로 정상 콘솔 환경에서 실행하고,
  
     성공 여부는 .exe 파일 생성 여부로 판단.
  
     오류 내용은 잠깐 뜨는 콘솔 창에서 확인.
